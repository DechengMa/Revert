Demonstrates the effect of UIVisualEffectView.

On top of the UIScrollView, there is a UIView which contains two visual effect views and a separator in the middle. The left view with Label A is a Visual Effect View with Blur. The right view with Label B is a Visual Effect View with Blur and Vibrancy.

The UIView moves around the UIScrollView along the path of bottom right, top right, bottom left and top left.
